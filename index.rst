PrestaShop Beslist.nl Shopping Cart Integration
===============================================


.. figure:: _static/images/logo.png
    :align: right
    :figwidth: 300px

    PrestaShop Beslist integration

The Beslist Shopping cart integration module is a module for PrestaShop. The module is available for sale at the official Prestashop addons store.

Installation
-----------
Buy the module aan via the `Prestashop addons store <https://addons.prestashop.com/en/marketplaces/23733-beslistnl-shopping-cart-integration.html>`_. When your PrestaShop instance is connected to your addons account you can now find the module in your backoffice.

You can also download the zip file from the addons website and upload it to your website via the interface. Managing modules works via your Prestashop administration, via the menu `Modules` > ` Modules & Services`. At the top right is a 'Add new module' button, where you can upload the zip file.

After successful installation you can see the :doc:`Module configuration page <configuration/module>`.
